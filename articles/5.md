组件和方法的按需加载方式，是不一样的

```
Vue.component(Button.name, Button)

import { Button, Message } from 'element-ui'

Vue.component(Button.name, Button)

```

#### 5-5 表单校验

不使用插件的时候，缺点？

1. 需要做大量的校验逻辑
2. 如果一个 form-item 有多个逻辑限制，需要多个限制判断

简化校验过程：

​	el-form 表单接受， rules  这个对象数组

​	通过key , 去配置 校验规则，每个 el-form-item 对应一个校验的 key
